# Contributing to IntentR

Thank you for your interest in contributing to IntentR! This document outlines areas where we need help and how you can get involved.

## What Help is Needed Right Now

We're actively looking for help with:

### Core Features

- **Approval Workflow Enhancements** - Improving the capability approval queue UI and adding batch approval functionality
- **Real-time Collaboration** - Extending WebSocket-based collaboration features for multi-user editing
- **Figma Integration** - Enhancing the design-to-code pipeline and component mapping
- **AI Chat Improvements** - Better context awareness and code generation accuracy

### Documentation

- **API Documentation** - Expanding OpenAPI/Swagger specs for all service endpoints
- **Tutorial Content** - Step-by-step guides for common workflows
- **INTENT Methodology Guides** - Practical examples of the Scaled Agile With AI approach
- **Architecture Decision Records** - Documenting key design decisions and rationale

### Integrations

- **Design Tool Plugins** - Extending beyond Figma to support Sketch, Adobe XD
- **CI/CD Pipelines** - GitHub Actions, GitLab CI templates for IntentR workflows
- **IDE Extensions** - VS Code and JetBrains plugins for specification editing
- **Project Management Tools** - Jira, Linear, and Notion integrations

### Tests

- **Unit Test Coverage** - Expanding Go service test coverage (target: 80%+)
- **React Component Tests** - Jest/Vitest tests for UI components
- **E2E Test Suites** - Playwright or Cypress tests for critical user journeys
- **BDD/Gherkin Scenarios** - Writing acceptance test scenarios linked to requirements
- **Performance Testing** - Load testing for collaboration and API services

## How to Get Started

1. Check the [README.md](README.md) for setup instructions
2. Review open issues labeled `good first issue` or `help wanted`
3. Read [CLAUDE.md](CLAUDE.md) for development methodology and code standards
4. Join discussions in GitHub Issues to propose ideas before starting work

## Questions?

Open an issue or start a discussion - we're happy to help you get started!
